# produce
Producing helpers for Ardour

# Actions

**Takes managment**

- **Take save**. Moves the rec-armed track's playlist to a new track, created with initially selected template and name

- **Takes merge**. Merges playlists on different tracks to one on the first selected track

**Sends management**

- **Add AUX Sends**. Add aux sends to last selected route for all selected routes except the last one
- **Set output**. Set output of selected routes to last selected route

**Groups managmenet**
- **Group Create**. Create group of selected tracks, create bus from template, set tracks outputs to bus
